##  Objetivo do Estudo
O objetivo é consolidar o uso de ferramentas essenciais em Machine Learning (Python, Pandas, Matplotlib) através de um estudo computacional completo. O foco está na organização e reprodutibilidade análise exploratória.

##  Estrutura do Repositório
* `notebooks/`: Contém o arquivo principal `.ipynb` com toda a análise e conclusões.
*`data/`: Base de dados real utilizada no projeto.
* `ambiente.yml`: Arquivo de configuração do Conda para replicar o ambiente de desenvolvimento.
* `README.md`: Este guia de descrição do repositório.
